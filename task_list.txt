1. створюємо папку, там де захочете
2. відкриваєте цю папку в провіднику
3. клікаєте в неї правою клавішою миші і обираєте пункт git bash here
4. клікаєте в git bash правою клавішою миші і вставляєте в нього наступну інструкцію:
    git clone https://github.com/valdemar0860/patriot_site-logika14-.git

5. переходите до склонованого репозиторія в провіднику і відркриваєте його в текстовому редакторі
6. відкриваєте файл register.html та додаєте у нього поля для отримання нікнейму та логіну, окрім паролю, додаючи для користувача інструкцію де яке поле.
7. відкриваєте файл user.html у цьому файлі, робите більш коректне відображення користувачів.
8. відкриваєте файл db_learn.py, створюєте у цьому файлі функцію-view для додавання користувачем свого рідного міста, якщо користувач не побачив його в списку (бажано додати перевірку для існування такого міста).
9. за додаткові бали можете реалізувати функцю логіну та додати лише користувачам зареєстрованим можливість додавати міста.
10. коректне відображення інформації про користувачів user_view.
